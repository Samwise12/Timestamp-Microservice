API Basejump: Timestamp microservice
User stories:
    1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: December 12, 2015)
    2) If it does, it returns both the Unix timestamp and the natural language from the date
    3) If it does not contain a date or Unix timestamp, it returns null for those properties

Example usage:

January%2012,%20201
-55794164400
